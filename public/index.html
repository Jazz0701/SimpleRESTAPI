REST API using Node.js with Express

API Endpoints

  • Users Endpoints
    • GET /users - Retrieve all users from database.
    • POST /users - Create a new user in the database.
    • PUT /users/:id - Update an existing user in the database.
    • DELETE /users/:id - Delete a user from the database.
  • Income Endpoints
    • GET /income - Retrieve all income records from database.
    • POST /income - Create a new income record in the database.
    • PUT /income/:id - Update an existing income record in the database.
    • DELETE /income/:id - Delete an income record from the database.
  • Expenses Endpoints
    • GET /expenses - Retrieve all expenses records from database.
    • POST /expenses - Create a new expenses record in the database.
    • PUT /expenses/:id - Update an existing expenses record in the database.
    • DELETE /expenses/:id - Delete an expenses record from the database.

Click the GET links to view data retrieved from Firebase. Use Postman or Thunder Client to perform POST, PUT, and DELETE operations.
